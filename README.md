## 📋 Overview

**ACRE** (AI Codebase Reverse Engineer) is a powerful, AI-driven security analysis platform that instantly analyzes repositories, source code, and websites. Built with a terminal-inspired cyberpunk aesthetic, ACRE provides comprehensive security insights, architecture visualization, and AI-powered code understanding.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔍 **Repository Analysis** | Analyze any GitHub repository in seconds — understand code structure, dependencies, and architecture |
| 🌐 **Website Scanning** | Scan websites for security vulnerabilities, trackers, and tech stack detection |
| 🛡️ **Security Analysis** | Detect vulnerabilities, exposed secrets, and security misconfigurations |
| 📦 **Dependency Audit** | Audit project dependencies for known vulnerabilities and outdated packages |
| 🤖 **AI Chat Assistant** | Ask questions about your codebase — get instant answers powered by AI |
| 📄 **Documentation Generator** | Auto-generate comprehensive documentation for any codebase |
| 📊 **Architecture Diagrams** | Visualize code architecture with auto-generated diagrams |
| ⚡ **Real-time Scanning** | Live scan progress with interactive status updates |

## 🏗️ Tech Stack

```
├── Frontend:    Next.js 15 (App Router), React 19, TypeScript
├── Styling:     Tailwind CSS 4, Framer Motion
├── Auth:        Clerk (Google OAuth, Magic Links)
├── UI:          Radix UI, Lucide Icons, CMDK
├── Charts:      ECharts, Mermaid.js
├── State:       Zustand, TanStack React Query
├── Backend:     Next.js API Routes, WebSocket Server
├── Database:    PostgreSQL (via Drizzle ORM)
├── Monorepo:    Turborepo
└── Deployment:  Vercel
```

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- npm 11+
- Clerk account (for authentication)
- PostgreSQL database (optional, for advanced features)

### Installation

```bash
# Clone the repository
git clone https://github.com/devtailor172004/freebuff.git
cd freebuff

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your Clerk keys and database URL

# Run development server
npm run dev
```

Visit `http://localhost:3000` to see the application.

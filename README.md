## 🔍 Overview

**ACRE (AI Codebase Reverse Engineer)** is an advanced security analysis platform that automatically scans repositories, source code, and web applications for vulnerabilities, misconfigurations, and security risks. Built with a terminal-inspired interface for security professionals.

## ✨ Capabilities

| Feature | Description |
|---------|-------------|
| 🔬 **Repository Analysis** | Deep code scanning for vulnerabilities and security flaws |
| 🌐 **Website Security** | Automated website reconnaissance and threat detection |
| 📊 **Dependency Auditing** | Supply chain risk assessment and vulnerability mapping |
| 🤖 **AI Chat Assistant** | Intelligent security companion for queries and guidance |
| 📐 **Architecture Diagrams** | Visual code structure mapping and dependency graphs |
| 📄 **Documentation Generation** | Automated security documentation creation |
